Project Description

Goal Statement

Finding a good place to study on campus can be a challenging task. When you only have an hour or two to between classes and have to walk back and forth between buildings looking for a place to study it can be easy to lose motivation. With Study Space you will be able to see where places to study on campus are and what resource they have available to students. Using user input along with analyzing trends we will be able to give real time updates on how full each area is. If you are looking for somewhere a little more private you will be able to reserve a study room.

Services

  • Find place on campus to study
  • Find a place near me
  • Easier to use than a map
  • Find group space
  • Find quiet (nap) space
  • Reserve a room
  • List Amenities (locks/whiteboards/outlets/wi-fi/printers)
  • Data Analysis
  • User Comments

Programming

This project is a web based application that will run on a linux (debian) machine.

Frontend

Development will be heavy on HTML, CSS, and JS. Use of the Semantic UI framework will make styling the site easier so that more focus can go into writing Javascript code that will make the site very interactive. Thymeleaf will be used as our web templating language.

Backend

Java is the language of choice, and the backend will be built on the Spring Framework. User contributed data will need to be stored and analyzed. We will be using MySQL for our database. Information about the users will need to be collected as well (ip address, gps coordinates, etc).

Typical Customer

College students looking for:

  • a place to study
  • a quiet place
  • a place to nap
  • a place to use and charge electronic devices
  • a group study space
